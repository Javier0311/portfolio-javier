## ✨ Key Features
- **Fully Responsive:** Engineered with a strict Mobile-First CSS architecture, ensuring a seamless and polished experience across smartphones, tablets, and desktops.
- **Modern UI/UX:** Clean, dark-themed interface with smooth scrolling, hover effects, and a highly readable layout.
- **Dynamic Content:** Structured React components modularizing the Hero, About, Skills, Projects, and Contact sections.
- **Accessibility Aware:** Proper use of semantic tags, `alt` attributes for images, and high-contrast color schemes.

## 🛠️ Technologies & Tools
- **Core:** React.js, JavaScript (ES6+), HTML5
- **Styling:** Custom CSS3 (Flexbox, CSS Grid, Media Queries)
- **Deployment & CI/CD:** Vercel
- **Version Control:** Git & GitHub
